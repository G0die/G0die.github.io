添加SSH key

访问Github的官网，注册Github的账号，常规填写name（建议自己特征明显的），e-mail（推荐谷歌邮箱），password。然后点击右上角小图标 →Sstting →SSH and GPG keys →SSH keys →New SSH key。把上述密匙全部粘贴。

验证Github邮箱

建立Github的仓库之前必须要先验证邮箱，我注册时候用的是qq邮箱，在这一步时总是收不到邮件，解决办法：

建立Github仓库

点击右上角+号 →New repository

  • Repository name一定要与自己的github的用户名相同，格式为用户用+.github.io

eg：我的github用户名是xinmiyp，则我的仓库名字xinmiyp.github.io

  • Description ：随意填
  • Initialize this repository with a README：勾选上。

然后点击Create repository。在新页面找到Settings →Launsh automatic page generator →Continue to layouts →Publish page

添加背景图片

1、首先找到一个背景图片放到 hexo（hexo工程文件）-> themes -> next -> source -> images 的路径下；

2、hexo（hexo工程文件）-> themes -> next -> source -> css -> _schemes -> Pisces（Mist和Muse也行），找到路径下的index.styl文件，在文件的最上方加上一代码 body { background:url(/images/backGround.jpg（这是你之前加的背景图片的名字）);} 就完事了。
